# Valentine Website

A simple one-page Valentine gift website.

## 1) Add your photo

Save your couple photo as:

`assets/our-photo.jpg`

## 2) Customize text

Edit these lines in `index.html`:

- `My Love`
- `Your Boyfriend`
- The letter paragraphs

## 3) Publish to GitHub Pages (`username.github.io`)

1. Create a repo on GitHub named exactly: `YOUR_USERNAME.github.io`
2. In this folder, run:

```bash
git add .
git commit -m "Create Valentine website"
git branch -M main
git remote add origin https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io.git
git push -u origin main
```

Your website link will be:

`https://YOUR_USERNAME.github.io/`

## Optional: Use a project repo instead

If you use repo name `valentine-for-her`, the link will be:

`https://YOUR_USERNAME.github.io/valentine-for-her/`
